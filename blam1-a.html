TAGS

Within that same HEK Directory, the TAGS folder, mirroing the DATA folder, holds all the imported tag data. This folder is where your GURELLIA TAG editor will be 100% of the time. TAGS are editable by GURELLIA which dictate game behavior, object properties, and so much more!

TAGS make up the entirety of what is Halo. TAG creation, especially as a newcommer, is a little daunting. Luckily, the developers of Halo, Bungie, crafted you the perfect blueprint to study off of... the game itself!

There are too much TAG values and properties.. and types.. that the best way to learn about what a TAG or its value does is to fuck around and find out. Also, refer to the incredibly useful webpage made by other, much more experienced modders in the community at The Reclaimers Liberary
